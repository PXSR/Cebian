你可以选择下面任意一种方式安装 Cebian。普通使用的话，建议优先从插件市场安装；如果想第一时间用上最新版本，可以从 Github Release 下载。

## 通过插件市场安装

Chrome、Edge 以及大多数 Chromium 系浏览器都可以从插件市场安装：

- [Chrome 插件市场](https://chromewebstore.google.com/detail/dnfmklmdbipiahdjiogaoemjaiodomdf)
- [Edge 插件市场](https://microsoftedge.microsoft.com/addons/detail/kakfgcmbiacmclkjboijakkhlhioopcd)

打开对应页面后，点击安装按钮即可。安装完成后，浏览器工具栏会出现 Cebian 图标，点击它就可以打开侧边栏。

> 商店版本需要等审核，所以可能会比 Github Release 晚一点更新。

## 通过 Github Release 安装

更新时也会在 Github 同步发布，你可以去 Releases 中下载最新版本：

[Cebian Github Releases](https://github.com/maotoumao/Cebian/releases)

安装步骤如下：

1. 下载 `cebian-{version}-chrome.zip`
2. 解压到一个固定目录
3. 打开浏览器扩展管理页面
4. 打开「开发者模式」
5. 点击「加载已解压的扩展程序」
6. 选择刚才解压出来的目录

> Release 里也会有 Firefox 包。如果你要在 Firefox 上手动安装，可以下载对应的 `firefox.zip`，不过这里先不展开 Firefox 的安装流程。

## 安装方式选择

- 普通使用：插件市场最省心
    - Edge 用户：优先用 Edge 插件市场
    - Chrome / Chromium 用户：优先用 Chrome 插件市场

- 想用最新版：Github Release 通常最快

## 安装后

安装完成后，点击浏览器工具栏里的 Cebian 图标打开侧边栏。第一次使用时，需要先配置一个 AI 提供商，然后就可以开始对话。
