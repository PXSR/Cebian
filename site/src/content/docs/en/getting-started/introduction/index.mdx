Cebian is an AI agent that lives in your browser sidebar. Unlike tools that open a chat box in a new tab, it stays right next to the page you're reading — reading page content, driving the browser, and extending its abilities through MCP and Skills.

<Callout type="tip">
You don't need to set up MCP and Skills on day one. Get a model working first, get a feel for the page-aware features, then go deeper.
</Callout>

## Core ideas

- **Local-first**: conversations, prompts, Skills, and attachments all live in your browser. Apart from the requests needed to reach your model provider, nothing is uploaded to any server.
- **Your own model**: enter any provider's API key — no account, no credits.
- **Extensible**: package workflows into Skills the model can pick up, and plug external tools in through MCP.

## What it can do

- Summarize the current page, long reads, docs, or PRs.
- Select anything and have the model summarize, translate, or rewrite it.
- Grab tables, lists, and prices from a page and turn them into JSON or Markdown.
- Record a set of actions once, then let the agent replay them.
