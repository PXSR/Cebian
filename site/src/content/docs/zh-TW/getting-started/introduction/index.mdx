Cebian 是一个常驻浏览器侧边栏的 AI Agent。和把对话框开在新标签页的工具不同，它就停在你正在看的页面旁边——读取页面内容、操作浏览器，并通过 MCP 与 Skill 扩展能力。

<Callout type="tip">
第一次用不必一上来就配 MCP 和 Skill。先把模型跑通、感受一下页面感知能力，再逐步加深。
</Callout>

## 核心特点

- **本地优先**：对话、Prompt、Skill、附件都存在浏览器本地。除了发给模型厂商的必要请求，不向任何服务器上传数据。
- **直连你自己的模型**：填入任意厂商的 API Key 即可，不需要账号、不需要积分。
- **可扩展**：用 Skill 把工作流打包成模型能自取的操作手册，用 MCP 把外部工具接进来。

## 它能做什么

- 总结当前网页、长文、文档或 PR。
- 选中任意内容，让模型总结、翻译、改写。
- 抓取页面里的表格、列表、价格，转成 JSON 或 Markdown。
- 录制一遍操作，让 Agent 之后自己重放。
